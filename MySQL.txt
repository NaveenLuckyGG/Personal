INNER JOIN:- Returns rows where there is a match in both tables.

SQL> SELECT e.empId, e.name, e.dept, d.depname FROM EMPLOYEE e
     INNER JOIN DEPT d ON e.dept = d.depname;

LEFT OUTER JOIN:- Returns all rows from the left table (EMPLOYEE), and matched rows from DEPT. If no match, NULL is returned.

SQL> SELECT e.empId, e.name, e.dept, d.depname FROM EMPLOYEE e
     LEFT JOIN DEPT d ON e.dept = d.depname;

RIGHT JOIN
Returns all rows from the right table (DEPT), and matched rows from EMPLOYEE.

SQL> SELECT e.empId, e.name, e.dept, d.depname FROM EMPLOYEE e
     RIGHT JOIN DEPT d ON e.dept = d.depname;

FULL OUTER JOIN (Not directly supported in MySQL)
You can simulate it using UNION of LEFT JOIN and RIGHT JOIN.

SQL> SELECT e.empId, e.name, e.dept, d.depname FROM EMPLOYEE e 
     LEFT JOIN DEPT d ON e.dept = d.depname
     UNION
     SELECT e.empId, e.name, e.dept, d.depname FROM EMPLOYEE e
     RIGHT JOIN DEPT d ON e.dept = d.depname;
CROSS JOIN
Returns Cartesian product (all combinations).

SQL> SELECT e.name, d.depname FROM EMPLOYEE e
     CROSS JOIN DEPT d;
